## 命令行

- 安装依赖：`npm install`
- 本地开发：`npm run dev`
- 发布生产：`npm run prod`

## 访问链接

- `http://localhost:3000`


## 目录

```
---
  |---- dist: 构建脚本
  |---- config: 配置文件
  |
  | |
  | |----base: 基础组件 如loading、轮播
  | |----components: 组件
  | |----entry: 入口js
  | |----fetch: 数据请求
  | |----router: 路由配置
  | |----static: 静态资源
  | |----template: 入口html，与entry一一对应
  | |----store: vuex配置
  | |
  |
  |---- server: node server 代码
---

```
